# WebRTC Streaming Server

基于 Mediasoup 的高性能 WebRTC 网络视频流解决方案。

## 文档

- [中文文档](./README.zh-CN.md)
- [English Documentation](./README.en-US.md)

## 快速概览

- 基于 Mediasoup 的 WebRTC 视频流媒体服务器
- 支持 FFmpeg / GStreamer Producer
- 支持 RTSP 协议和 USB 摄像头
- 支持音频播放和语音对讲
- 支持网页录制和后台分片录制
- 集成多种视频播放器
- Web 端基于 React 18/19
- 最低支持 Ubuntu 16.04 / Node.js 16
